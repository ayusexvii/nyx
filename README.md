# Project NYX

**Nyx** — Greek primordial goddess of night. Born from Chaos. Feared by Zeus.

A modular bioinformatics platform for multi-domain sequence analysis.

## Philosophy

> "Before light, there was Nyx. Before answers, there was darkness. Work in the shadows others fear to explore."

## Domains

- Metagenomics (current)
- Precision Medicine (next)
- CRISPR (future)
- Cloning (future)

## Architecture

- Core engine (never changes)
- Domain plugins (add over time)
- CLI tool: `nyx`
